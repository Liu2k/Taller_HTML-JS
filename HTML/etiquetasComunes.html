Etiquetas Comunes en HTML

Aprendamos sobre las etiquetas más utilizadas en HTML

¿Qué son las etiquetas en HTML?

Las etiquetas son la base de HTML y se utilizan para estructurar y definir el contenido en una página web. A continuación, veremos algunas de las etiquetas más comunes.

Lista de Etiquetas Comunes

  • <html>: Define el documento HTML.
  • <head>: Contiene información sobre la página.
  • <title>: Define el título de la página.
  • <body>: Contiene el contenido visible de la página.
  • <h1> a <h6>: Encabezados de diferentes niveles.
  • <p>: Define un párrafo.
  • <ul>: Lista desordenada.
  • <li>: Elemento de lista.
  • <footer>: Define el pie de página.
